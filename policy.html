политика использования

1. общие положения

сервис предоставляет инструменты для защиты интернет-соединения и маршрутизации трафика. использование сервиса означает согласие с данной политикой.

2. предоставляемые услуги

  • доступ к прокси-сервису
  • выдача vless-ключей для подключения
  • доступ к серверной инфраструктуре через клиентские приложения

3. условия использования

  • один пользователь может использовать до 5 устройств
  • передача доступа третьим лицам ограничена
  • массовое распространение конфигураций запрещено

4. telegram-прокси

  • при активной подписке предоставляется telegram-прокси
  • сервис заботится о стабильной связи с клиентами
  • telegram-прокси является дополнительной услугой к vless-прокси
  • прокси работает только при активной подписке
  • предназначен для поддержания постоянного доступа к telegram

5. ограничения

  • запрещён шейринг telegram-прокси
  • запрещено использование сервиса для незаконной деятельности
  • запрещены попытки перегрузки сервиса или обхода ограничений

6. безопасность

  • используются современные методы шифрования
  • сервис направлен на защиту соединения пользователя
  • поддерживается стабильная и безопасная работа

7. ответственность

  • пользователь самостоятельно несёт ответственность за использование сервиса
  • сервис не отвечает за действия пользователей

8. изменения условий

  • условия могут обновляться без предварительного уведомления
  • актуальная версия всегда доступна на сайте
